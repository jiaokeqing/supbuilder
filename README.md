### 上手指南
###### 开发前的配置要求
###### 开发环境
+ JDK 1.8
+ Mysql 5.7
+ Maven 3.6.3
+ Redis 3.2
###### 开发工具
+ IDEA
+ PostMan


###### **安装步骤**

1. Clone the repo

```sh
git clone https://github.com/jiaokeqing/supbuilder.git
```
